OSI MODEL
The Open Systems Interconnection, or known as OSI, model is made up of 7 layers, or levels. The first level, starting at the bottom, is the physical layer and this is the physical connections between devices (i.e. ethernet cables, wireless/WIFI, fiber optic). The second layer, data link, handles data in and out of a physical connection and the devices who lie at the data link layer recognize each other through MAC addresses (i.e. switch). The third layer, network, is responsible for routing packets to the right destination and devices who lay at this layer recognize each other through IP addresses (i.e. router). The fourth layer is the transport layer that handles the delivery of data from host points to host points (i.e. TCP and UDP). The fifth layer, session, which manages the set period of data transfer to occur. The sixth layer, presentation, formats the data in a way that can be read by the application or protocol at the application layer (i.e. decrypting encrypted data, decompressing data). The seventh and final layer is the application layer that serves as a specification of protocol that is used between hosts when communicating on the network or internet (i.e. HTTP, HTTPS, SSH, SMTP).
